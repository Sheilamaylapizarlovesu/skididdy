ICT 4th Quarter Lessons

Events are actions triggered by the user or browser such as clicking a button, moving the mouse, or loading a webpage. Event handlers connect these actions to JavaScript functions so the webpage reacts to user interaction. Common events include onclick, onmouseover, onload, and onsubmit. Pop-up boxes are windows used to interact with users. The alert() box displays warnings or messages, prompt() asks the user for input, and confirm() asks the user to choose between OK or Cancel.

Conditional statements allow programs to make decisions based on conditions. The if statement executes code when a condition is true. If-else provides an alternative action when the condition is false. If-else-if checks multiple conditions sequentially. Nested if statements contain another if inside a condition. The switch statement is used when a variable can have many possible values and each value runs a different set of instructions.

A function is a reusable block of code that performs a specific task when called. Functions help organize programs and prevent repeating code. They are defined using the keyword function followed by a name and parentheses containing parameters. Parameters provide additional information for processing. Functions may also use the return statement to send results back to the program.

Loops repeat a block of code until a condition is met. The for loop runs a set number of times. The while loop continues while a condition remains true. The do-while loop runs the code at least once before checking the condition. The for-in loop iterates through properties of objects or arrays. Break stops the loop immediately, while continue skips the current iteration and proceeds to the next one.

Strings represent text and contain properties and methods for manipulating characters. Examples include length, replace(), and toUpperCase(). Arrays store multiple values in a single variable and allow operations such as push(), pop(), sort(), and join(). The Date object is used to access and manipulate the current date and time in JavaScript applications.

JavaScript can execute actions after a specific time using timing functions. setTimeout() runs code after a delay while clearTimeout() cancels the timer. The Math object provides built-in constants and functions used for calculations such as Math.random(), Math.round(), Math.sqrt(), Math.max(), and Math.pow().
